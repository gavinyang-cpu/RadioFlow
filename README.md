# RadioFlow

macOS app for recording internet radio and transcribing with AI.

## Features

- 📻 **Stream Radio** - Listen to internet radio stations
- 🎙️ **Record** - Record radio broadcasts
- 🤖 **Transcribe** - Generate transcripts using AI
- 📊 **Manage** - Organize recordings and transcriptions

## Requirements

- macOS 14.0+
- Xcode 15+

## Getting Started

1. Clone the repo
2. Open `RadioFlow.xcodeproj` in Xcode
3. Build and run

## Usage

1. Select a radio station from the sidebar
2. Click play to start streaming
3. Click record to save the broadcast
4. Use the AI transcription feature to generate scripts

## API Configuration

For transcription, configure your AI API endpoint in Settings:
- API Endpoint
- API Key
